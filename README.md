# Retail Sales Data Analysis using SQL
📌 Project Overview

Retail businesses generate huge volumes of sales data every day. Analyzing this data helps companies understand customer behavior, identify sales trends, and make better business decisions.

This project focuses on analyzing Retail Sales data using SQL to extract valuable insights about product performance, customer purchasing patterns, and overall sales trends.

By performing structured queries on the dataset, the project demonstrates how SQL can be used as a powerful tool for data analysis in real-world business scenarios.

The analysis includes:

Total sales analysis
Product performance evaluation
Revenue calculations
Monthly sales trends
Customer purchasing patterns
This project is designed to demonstrate SQL skills required for Data Analyst roles.

# Project Objectives

The main objectives of this project are:
Analyze retail sales data using SQL queries
Calculate overall sales performance
Identify top-selling products
Track monthly revenue trends
Understand customer buying patterns
Measure order cancellation rate
Extract actionable business insights
